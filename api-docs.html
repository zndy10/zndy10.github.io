RSA 签名认证

鉴权
本平台采用 RSA 签名认证机制，无需登录账号密码，使用私钥对请求签名，平台使用公钥验证签名有效性。
签名 Header 参数
参数Header 名称必传说明
app_idX-App-Id是由平台分配的 App ID
timestampX-Timestamp是当前时间戳（毫秒），有效期 ±5 分钟
nonceX-Nonce是随机字符串（建议 UUID）
signatureX-Signature是RSA 签名值，Base64 编码
待签名字符串格式
待签名字符串 = app_id + "|" + timestamp + "|" + nonce + "|" + normalizedRequestBody

Body 规范化: JSON key 按字典序排序，紧凑格式输出
GET请求 body 为空，末尾为空字符串
代码示例
Java
Python
Node.js
String signStr = APP_ID + "|" + timestamp + "|" + nonce + "|" + requestBody;
byte[] keyBytes = Base64.getDecoder().decode(PRIVATE_KEY);
PrivateKey privateKey = KeyFactory.getInstance("RSA")
    .generatePrivate(new PKCS8EncodedKeySpec(keyBytes));
Signature sig = Signature.getInstance("SHA256withRSA");
sig.initSign(privateKey);
sig.update(signStr.getBytes("UTF-8"));
String result = Base64.getEncoder().encodeToString(sig.sign());
from Crypto.PublicKey import RSA
from Crypto.Signature import pkcs1_15
from Crypto.Hash import SHA256
import base64

sign_str = f"{APP_ID}|{timestamp}|{nonce}|{request_body}"
private_key = RSA.import_key(base64.b64decode(PRIVATE_KEY))
h = SHA256.new(sign_str.encode('utf-8'))
signature = base64.b64encode(pkcs1_15.new(private_key).sign(h)).decode()
const crypto = require('crypto');
const signStr = `${APP_ID}|${timestamp}|${nonce}|${requestBody}`;
const privateKey = Buffer.from(PRIVATE_KEY, 'base64');
const sign = crypto.createSign('RSA-SHA256');
sign.update(signStr);
const signature = sign.sign(privateKey, 'base64');

平台回调验签机制

鉴权
密钥关系
场景签名方验签方您需要持有
调用平台 API您（您的私钥）平台（您的公钥）您的私钥
接收平台回调平台（平台私钥）您（平台公钥）平台公钥
回调 HTTP Headers
参数Header 名称说明
app_idX-App-Id您的 App ID
timestampX-Timestamp时间戳（毫秒）
nonceX-Nonce随机字符串
signatureX-Signature平台签名
eventX-Event回调事件类型
sequenceIdX-Sequence-Id序列号（幂等处理）
回调重试机制
失败后自动重试，最大 3 次，间隔递增（1s、2s、3s）
